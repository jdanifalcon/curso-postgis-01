# Introducción a PostGIS: Datos Espaciales con SQL

Curso en línea · 12 al 16 de agosto 2026 · 10 horas

## Descripción

Curso introductorio de PostGIS donde aprenderás a crear y administrar bases de datos espaciales en PostgreSQL, ejecutar consultas y análisis espaciales con SQL e integrar tus resultados con QGIS.

## Temario

| Sesión | Día | Tema |
|--------|-----|------|
| 1 | Miércoles | Fundamentos: PostgreSQL y PostGIS |
| 2 | Jueves | Geometrías y datos espaciales |
| 3 | Viernes | Consultas espaciales |
| 4 | Sábado | Geoprocesamiento y visualización |
| 5 | Domingo | Proyecto integrador y buenas prácticas |

## Incluye

- Acceso a las 5 sesiones en vivo (Google Meet)
- Grabaciones de todas las sesiones
- Repositorio con scripts SQL y datos de práctica
- Constancia digital verificable con código QR

## Requisitos

- Manejo básico de computadora
- No se requiere experiencia en SQL ni bases de datos
- Software: PostgreSQL 16+, PostGIS 3.4+, pgAdmin 4, QGIS 3.34+
